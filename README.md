拼音分割 or 拼音分词
======

用来把没有空格的拼音字符串分割成有空格的拼音字符串。

```
Example:
>> PinyinSplit.split("wolegequ")
=> wo le ge qu

Arguments:
	source: (String)

Example:
>> PinyinSplit.splitArray("wolegequ")
=> ["wo","le","ge","qu"]

Arguments:
  source: (String)
```
